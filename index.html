VIE entitypreview widget

VIE entitypreview uses the VIE.find service method to make entitypreview suggestions.
The VIE.find method can query different backend or frontend data sources.
This demo goes to the default Apache Stanbol backend.

Usage

var vie = new VIE();
vie.use(new vie.StanbolService({
    url : "http://dev.iks-project.eu:8081",
    proxyDisabled: true
}));

$('.demo [resource]')
.entitypreview({
    vie: vie,
});

In case you only need the preview HTML snippet, you can do this:

jQuery('<div/>')
.entitypreview({
    vie: vie,
    uri: "http://dbpedia.org/resource/Salzburg"
})
.entitypreview('getPreviewHtml',
    function(html){
        console.info('html', html);
        // or whatever you like to do with it.
    }
);
